# Corrida dos Sapos com Threads

## Objetivo
Projeto em Java para simular uma corrida entre sapos, onde cada sapo é executado em uma thread separada e o progresso é exibido no console até a definição do vencedor.

## Tecnologias utilizadas
- Java 17
- Maven
- IntelliJ IDEA
- Git e GitHub

## Como rodar
1. Clone o repositório.
2. Abra o projeto na sua IDE Java.
3. Execute a classe `Main`.
